Document

HTML - документ - это обычный текстовый документ, может быть создан как в обычном текстовом редакторе (Блокнот),
так и в специализированном, с подсветкой кода (Notepad++, Visual Studio Code и т.п). HTML - документ имеет расширение .html.

HTML - документсостоит из дерева HTML - элементов и текста. Каждый элемент обозначается в исходном документе
начальном (открывающим) и конечным (закрывающим) тегом (за редким исключением). Начальный тег показывает,
где начинается элемент, конечный - где заканчивается. Закрывающий тег образуется
путем добавления слэша / перед именем тега: ‹имя тега›...›/имя тега›.
Между начальным и закрывающим тегами находится содержимое элемента - контент.
Например:
1. ‹ p › Контент ‹ /p › - Создает параграф
2. ‹ b › Контент ‹ /b › - Делает полужирное начертание
3. ‹ i ‹ Контент ‹ /i › - Курсивное начертание
